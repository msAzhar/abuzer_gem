# AbuzerGem

Abuzer is awesome! :D

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'abuzer_gem'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install abuzer_gem

## Usage

AbuzerGem::Abuzer.new("Tom").greeting
Pio, Tom! I'm Abuzer!
